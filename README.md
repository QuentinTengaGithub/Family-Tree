# Family Tree

An interactive **family tree visualization** application built with **Vue.js**, **D3.js** and **Firebase**. The project allows users to create, explore and manage family relationships through an intuitive and dynamic graphical interface.

# Overview

Family Tree is a frontend application that visualizes complex family relationships through an interactive graph.

Instead of displaying static genealogy information, the application dynamically renders family members and their relationships, allowing users to navigate through generations with a fluid and intuitive experience.

The project focuses on data visualization, scalable architecture and interactive user interfaces.

# Installation

Clone the repository

```bash
git clone https://github.com/QuentinTengaGithub/Family-Tree.git
```

Navigate to the project

```bash
cd Family-Tree
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run serve
```
